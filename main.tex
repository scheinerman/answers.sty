answers in a \LaTeX\ book. 

\section{Arithmetic}

This first section concerns basic arithmetic.

\begin{exercises}

\item How much is $2+2$?
  \begin{answer}
    $2+2= 4$ 
  \end{answer}

\item This problem has no answer.

\item How much is $3\times 3$? 
  \begin{answer}
    $3\times3 = 9$. 
  \end{answer}

\end{exercises}
